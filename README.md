project- Multithreaded Chat Application
A console-based multithreaded chat application developed in Java using sockets.
It allows multiple clients to connect to a server and chat with each other simultaneously.

Features-
1.Multiple clients can connect at the same time
2.Each client runs on a separate thread
3.Messages are broadcast to all connected clients
4.Server runs continuously and handles connections

Technologies Used-
1.Java
2.Java Sockets
3.Multithreading
4.Console I/O

How to Run-
1. Start the Server
Open a terminal and run: java ChatServer
You should see:Chat Server started...
2. Start Clients
   Open multiple terminals and run: java ChatClient
   
Each client will enter a username and start chatting.
